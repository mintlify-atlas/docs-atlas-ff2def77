Get your WalletSync API running locally and make your first API call in minutes.

## Quick Setup with Docker Compose

The fastest way to get started is using Docker Compose, which sets up both the application and PostgreSQL database automatically.

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/alemadorran/walletsync.git
    cd walletsync
    ```
  </Step>

  <Step title="Start the services">
    Use Docker Compose to start both the application and database:
    
    ```bash
    docker-compose up -d
    ```
    
    <Info>The application will be available at `http://localhost:8080` once the containers are running.</Info>
  </Step>

  <Step title="Verify the API is running">
    Test that the API is accessible:
    
    ```bash
    curl http://localhost:8080/api/categories
    ```
    
    You should receive an empty array `[]` since no categories exist yet.
  </Step>
</Steps>

## Make Your First API Call

Now that WalletSync is running, let's create your first expense category and add a transaction.

<Steps>
  <Step title="Create a category">
    Categories help organize your transactions. Let's create a "Groceries" category:
    
    ```bash cURL
    curl -X POST http://localhost:8080/api/categories \
      -H "Content-Type: application/json" \
      -d '{
        "categoryName": "Groceries",
        "categoryType": "EXPENSE"
      }'
    ```
    
    <Accordion title="Expected response">
      ```json
      {
        "id": 1,
        "categoryName": "Groceries",
        "categoryType": "EXPENSE"
      }
      ```
    </Accordion>
  </Step>

  <Step title="Add a transaction">
    Create a payment movement for your new category:
    
    ```bash cURL
    curl -X POST http://localhost:8080/api/payments/santander \
      -H "Content-Type: application/json" \
      -d '{
        "operationDate": "2024-03-15",
        "operationLabel": "Compra Mercadona",
        "amount": -45.50,
        "balance": 1234.50
      }'
    ```
    
    <Accordion title="Expected response">
      ```json
      {
        "id": 1,
        "operationDate": "2024-03-15",
        "operationLabel": "Compra Mercadona",
        "amount": -45.50,
        "balance": 1234.50
      }
      ```
    </Accordion>
  </Step>

  <Step title="Create an association rule">
    Set up automatic categorization by associating keywords with categories:
    
    ```bash cURL
    curl -X POST http://localhost:8080/api/associations \
      -H "Content-Type: application/json" \
      -d '{
        "associationWord": "Mercadona",
        "categoryId": 1
      }'
    ```
    
    Now any transaction containing "Mercadona" will automatically be categorized as "Groceries".
  </Step>

  <Step title="View categorized transactions">
    Retrieve all transactions with their assigned categories:
    
    ```bash cURL
    curl http://localhost:8080/api/payments/santander/inCategories
    ```
    
    <Accordion title="Expected response">
      ```json
      [
        {
          "category": {
            "id": 1,
            "categoryName": "Groceries",
            "categoryType": "EXPENSE"
          },
          "santanderPaymentMovements": [
            {
              "id": 1,
              "operationDate": "2024-03-15",
              "operationLabel": "Compra Mercadona",
              "amount": -45.50,
              "balance": 1234.50
            }
          ]
        }
      ]
      ```
    </Accordion>
  </Step>
</Steps>
